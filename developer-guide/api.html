Cloud AUD wallet API¶

POST /wallets¶

Create a new wallet.

Create a new CAUD wallet.

Status Codes:

  • 201 Created – Successful Response

  • 422 Unprocessable Entity – Validation Error

PATCH /wallets¶

Update an existing wallet.

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error

GET /wallets/{unique_id}¶

Get Wallet

Parameters:

  • unique_id (string)

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error

POST /wallets/{unique_id}/verify_pin¶

Verify passcode for an existing wallet.

Parameters:

  • unique_id (string)

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error

GET /wallets/{unique_id}/balance¶

Get balance for Wallet

Get the current balance of a wallet.

Args:

unique_id: unique wallet id

Returns:

SendCaudResponseModel: See schema below

Parameters:

  • unique_id (string)

Status Codes:

  • 200 OK – Successful Response

  • 422 Unprocessable Entity – Validation Error

POST /wallets/{unique_id}/send_caud¶

Send caud to a Wallet

Send caud to a destination wallet address.<br/> __Note that amount is in dollars and cents__

Args:

send_caud_model (SendCaudModel): See schema

Returns:

SendCaudResponseModel: Send caud response

Parameters:

  • unique_id (string)

  • 200 OK – Successful Response
